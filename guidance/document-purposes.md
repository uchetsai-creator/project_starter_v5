## How to use this file

Load two files:
1. `document-purposes-common.md` — entries that apply to all project types
2. The file for your declared project type (see table below)

**Mixed / Hybrid types:** load `document-purposes-common.md` + all per-type files matching your declared types.

Example: `Project Type: Data Pipeline + Web App` → load `document-purposes-common.md` + `document-purposes-data-pipeline.md` + `document-purposes-web-app.md`.
